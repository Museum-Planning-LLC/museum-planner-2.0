# museum-planner-2.0

Source and documentation for **Museum Planner** (**museumplanner.org**): positioning, voice, static homepage, essays, and repo hygiene. It sits alongside **Museum Planning, LLC** (**museumplanning.com**) as the **practitioner-facing** side of the practice—community, courses, and long-form writing—not the commercial checkout for engagements.

**Repo home:** **Museum Planning LLC** on GitHub — [`Museum-Planning-LLC/museum-planner-2.0`](https://github.com/Museum-Planning-LLC/museum-planner-2.0). **Not** the Walhimer Studio artwork site (**walhimer.github.io**).

## Quick positioning

- **museumplanning.com** — **Museum Planning, LLC**: hire for feasibility, master planning, historic buildings, opening day.
- **museumplanner.org** — **Museum Planner**: community (Skool), AI-driven courses, essays, and archive; **Hire Mark** and practice links point back to the firm.
- **Visuals** — Homepage = v3 dark system in `index.html`; long-form essays may use the light **Practitioner Intelligence** template (see `genz_town_square_article.html`). Details in `STYLE-GUIDE.md`.
