## Quick positioning

- **museumplanning.com** — sales vehicle for master planning and historic-building-to-museum work (Kit, PR, campaigns).
- **Museum Planner** — depth, books, and future courses; links **from** Museum Planning for readers who want more, without replacing paid engagements.
- **Visuals** — one family: same logo system and palette on both sites; see `STYLE-GUIDE.md` → *Visual identity*.
